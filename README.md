# Airbnb API

REST & GraphQL API of the Airbnb Clone using Django REST Framework and Graphene GraphQL

## API Actions

- [x] JWT Authentication
- [x] Login
- [x] Create Account
- [x] See Profile
- [x] Edit Profile
- [x] Add / Remove Room from Favorites
- [x] List Rooms
- [x] See Room
- [x] Create Room
- [x] Edit Room
- [x] Delete Room
- [x] Search Rooms
